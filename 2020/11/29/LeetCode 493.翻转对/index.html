LeetCode 493.翻转对

LeetCode 493.翻转对

题目

原题链接

给定一个数组 nums ，如果 i < j 且 nums[i] > 2*nums[j] 我们就将 (i, j) 称作一个重要翻转对。

你需要返回给定数组中的重要翻转对的数量。

示例 1:

输入: [1,3,2,3,1]
输出: 2

示例 2:

输入: [2,4,3,5,1]
输出: 3

注意:

​ 给定数组的长度不会超过50000。
​ 输入数组中的所有数字都在32位整数的表示范围内。

思路

输入: [1,3,2,3,1]
输出: 2

res1: 1 3 2 3 1 （依据题意每个*2
res2: 2 6 4 6 2

这道题目就变成了，给定一个位置pos，res2[pos-n] 比 res1[pos]大的值的数量

那么朴素的思路就是，建一个哈希表，然后从后往前遍历，对于每个res1[i],只需要遍历哈希表，找出比res1[i]小的个数，记录下来，然后在哈希表中存入res2[i]。

首先，数的大小范围是32位整数，包括负数，导致自己创造哈希表需要加上一个偏移量，还有数组过大的问题。×2还会导致整数溢出。

每次都要遍历一次哈希表，时间复杂度过高。如果使用前缀和，会导致插入操作的时间复杂度为O(n);

离散化

可以使用一个unordered_map进行离散化数据。由于需要比较的是两个数组的所有值，所以需要吧两个数组合在一起，排序，编号。

线段树

线段树是用于求前缀和，插入的时间复杂度是O(logN),求前缀和的时间复杂是O(logN)。

class Solution {
public:
    int n=1;
    int fa[100010];
    int lowbit(int x){
        return x&-x;
    }

    int ask(int x){
        int res=0;
        while(x>0){
            res+=fa[x];
            x-=lowbit(x);
        }
        return res;
    }

    void update(int x,int k){
        while(x<=n){
            fa[x]+=k;
            x+=lowbit(x);
        }
    }

    int reversePairs(vector<int>& nums) {
        //前缀和和树状数组
        vector<long long> tem;
        for(auto& e:nums){
            tem.push_back(e);
            long long k=e;
            tem.push_back(k*2);
        }
        sort(tem.begin(),tem.end());
        unordered_map<long long,int> res;
        for(auto& e:tem){
            res[e]=n++;
        }
        int m=nums.size();
        int cur=0;
        for(int i=m-1;i>=0;i--){
            long long k=(long long)nums[i]*2;
            cur+=ask(res[nums[i]]-1);
            update(res[k],1);
        }
        return cur;
    }
};
