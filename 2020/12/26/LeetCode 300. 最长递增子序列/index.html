LeetCode 300.最长递增子序列

300. 最长递增子序列

题目

LeetCode

给你一个整数数组 nums ，找到其中最长严格递增子序列的长度。

子序列是由数组派生而来的序列，删除（或不删除）数组中的元素而不改变其余元素的顺序。例如，[3,6,2,7] 是数组 [0,3,1,6,2,2,7] 的子序列。

动态规划

dp[i] 表示 nums数组中0 ~ i之间的最大上升序列

状态转移方程:

​
$$
dp[i] = max(dp[j]) + 1, 0 <j< i-1, nums[j] < nums[i]
$$

class Solution {
public:
    int lengthOfLIS(vector<int>& nums) {
        int n = nums.size();
        vector<int> dp(n);
        int imax = 0;
        for(int i = 0 ;i < n; i++){
            dp[i] = 1;
            for(int j = 0; j < i; j++){
                if(nums[j] < nums[i]) {
                    dp[i] = max(dp[i], dp[j] + 1);
                }
            }
            imax = max(imax, dp[i]);
        }
        return imax;
    }
};

$$
时间复杂度为： O(n^2)
$$

剪枝

暴力＋剪枝

class Solution {
public:
    int imax=0;
    int getLength(vector<int>& nums,vector<int>& result,int position){
        if(position<result.size()){
            if(result[position]){
                return result[position];
            }
            for(int i=position+1;i<nums.size();i++){
                if(nums[i]>nums[position]){
                    result[position]=max(result[position],getLength(nums,result,i)+1);
                }
            }
            return result[position]?result[position]:1;
        }
        else{
            return 0;
        }
    }
    int lengthOfLIS(vector<int>& nums) {
        vector<int> result(nums.size(),0);
        for(int i=0;i<nums.size();i++){
            imax=max(imax,getLength(nums,result,i));
        }
        return imax;
    }
};

贪心+二分

这个方法其实很巧妙

如果我们要使上升子序列尽可能的长，则我们需要让序列上升得尽可能慢，因此我们希望每次在上升子序列最后加上的那个数尽可能的小。

下面的是官方的解析，官方还是一如既往的严谨。

作者：LeetCode-Solution
链接：https://leetcode-cn.com/problems/longest-increasing-subsequence/solution/zui-chang-shang-sheng-zi-xu-lie-by-leetcode-soluti/
来源：力扣（LeetCode）

其实简单来说，将不重复的数字换入到返回的结果数组res中，由于只是交换，并不改变其长度，并且使这个序列上升的很慢。虽然这个序列有时候是不合法的，但是，如果res中最后一个数组被替换，那么就变为合法的了。

利用二分法，可以在logN的时间复杂度中找到需要找到的位置： 第一个等于或者大于nums[i]

class Solution {
public:
    int lengthOfLIS(vector<int>& nums) {
        int n = nums.size();
        if(!n){
            return 0;
        }
        vector<int> res;
        res.push_back(nums[0]);
        int len = 1;
        for(int i = 1; i < n; i++){
            if(res.back() < nums[i]){
                res.push_back(nums[i]);
                len ++;
            }
            else if(res.back() > nums[i]){
                //二分查找
                int left = 0;
                int right = len - 1;
                while(left <= right){
                    //找到第一个比num[i]大的数
                    int mid = left + (right - left)/2;
                    if(res[mid] >= nums[i]) {
                        right = mid - 1;
                    }
                    else{
                        left = mid + 1;
                    }
                }
                if(res[left] != nums[i]) {
                    res[left] = nums[i];
                }
            }
        }
        return len;
    }
};
